# add_kanban.shを更新する
## 要望
add_kanban.shでファイルの作成をしているが、テンプレート機能が欲しい。
テンプレートは1種類固定で良い。

## 目的
`/kanban` コマンドで目的に該当する項目を必須にするとしたので、確実に書くようにするため。

## テンプレート
```md
# title
## 要望

## 目的

```

## プラン

- `bin/add_kanban.sh` の `touch "$filename"` を heredoc によるテンプレート書き込みに置き換える
- テンプレート内の `# title` 部分には引数の `$title` を展開する
- 変更は1ファイル・1箇所のみ
